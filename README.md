# miniGame-avoidShape


Simple Avoidance Game

-GUI: Win32api
-Language: C++
-소요시간: 2일

win32api를 응용해서 만들어본 간단한 사각형 피하기 게임


창크기, 떨어지는 속도, 장애물 나오는 주기 모두 변경가능
1/12 충돌처리구현
방향키, ESC 입력시 게임 종료
